Select Neighborhoods From The List or Map

Real Estate Explorer Instructions

You can select one or multiple neighborhoods either using the list or the map.

How to use Map
- On the map the area shaded light gray is in Ames City without any real estate sales info.
- The selected neighborhood are shaded with dark purple.
- The area colored with light purple is neighborhood with data which has not been selected to be included.
- To select a neighborhood from the map, click on it. The second click will unselect the neighborhood.
How to use List
- To select a Neighborhood from the list just click on it.
- To select multiple neighborhoods from the list just on Windows Ctrl + click on Mac use cmd + click.
- To select all neighborhoods on Windows Ctrl + a on Mac use cmd + a.
Charts
- There are different analytical charts available. Click on each tab item to checkout the chart. The charts will be adjusted to the selected neighborhoods.
